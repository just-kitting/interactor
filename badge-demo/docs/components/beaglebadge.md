# BeagleBadge

This submodule contains the design files for BeagleBadge

Upstream: https://openbeagle.org/beaglebadge/beaglebadge
Mirror: https://github.com/beagleboard/beaglebadge
Our fork: https://github.com/just-kitting/BeagleBadge

Modifications to improve documentation for BeagleBadge should be performed directly to this repository suitable for all BeagleBadge users.

Commit locally and the maintainer will provide feedback.

BadgeSnake will use this submodule repo to understand the design of BeagleBadge.
