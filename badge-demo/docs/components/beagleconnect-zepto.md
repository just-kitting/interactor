# BeagleConnect Zepto

This submodule contains the design files for BeagleConnect Zepto

Upstream: https://openbeagle.org/beagleconnect/zepto
Mirror: https://github.com/beagleboard/beagleconnect-zepto

Modifications to improve documentation for BeagleConnect Zepto should be performed directly to this repository suitable for all BeagleConnect Zepto users.

Commit locally and the maintainer will provide feedback.

BadgeSnake will use this submodule repo to understand the design of BeagleConnect Zepto.
